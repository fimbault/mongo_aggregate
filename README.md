# mongo_aggregate
basic example of aggregation, using mongoose

# setup
set MONGO variable in .env file, like 'MONGO=mongodb+srv://USER:PASSWORD@cluster0.o4gnc.mongodb.net/DATABASE?retryWrites=true&w=majority'

# start
'npm install'

'node main.js'

# other utilities

To test your queries:
- mongo shell
- mongodb compass
